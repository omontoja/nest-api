## **Projeto api-lider**

**Dependencias:**

- Nest/Typescript
- Sequelize

  **Setup do projeto:**

  **1-** Na raiz do projeto execute o comando:

       npm install

  **2-** No arquivo `database.providers.ts`, insira as credenciais do seu banco de dados SQL

  **3-** Iniciar o servidor com o comando:

       npm run start:dev

  **Testes:**

Para testar os end points da api, importe o arquivo presente na pasta "test_api" no seu programa de requisições http preferido
